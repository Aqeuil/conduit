# 模块划分

|      文件夹      |              说明              |
|:-------------:|:----------------------------:|
|  app-config   |             组件配置             |
|     core      | 最核心的运行时模块，实现环形调度、流程片段执行与任务编排 |
|     docs      |              文档              |
| observability |           实现系统可观测性           |
|      pkg      |              工具              |
|   protocols   |           处理多协议的转换           |
|    server     |             外部接口             |
|    console    |        提供系统管控界面与 API         |
